# CrowdSimulationLab

## Instalação

Siga os passos abaixo para configurar o ambiente local após clonar o repositório:

1. **Criar o ambiente virtual:**
   ```bash
   python3 -m venv venv
   ```

2. **Ativar o ambiente:**
   No macOS/Linux:
   ```bash
   source venv/bin/activate
   ```

3. **Instalar dependências:**
   Com o venv ativo, instale as bibliotecas necessárias:
   ```bash
   pip install -r requirements.txt
   ```

## Como Rodar

Após a instalação, sempre que desejar executar o laboratório, garanta que o ambiente virtual esteja ativo e execute:

```bash
python main.py
```
